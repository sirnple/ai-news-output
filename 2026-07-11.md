### 1. Remember When It Matters: Proactive Memory Agent for Long-Horizon Agents
> 来源：papers | 标签：Agent

**中文摘要**  
本文提出了一种面向长周期任务的自适应记忆代理（Proactive Memory Agent），其核心贡献在于通过主动记忆管理机制显著提升智能体在长期交互中的任务完成效率与准确性。方法上，该代理基于事件重要性动态评估与记忆检索优先级排序，在关键决策时刻主动激活相关历史经验，而非被动等待查询；关键发现表明，该方法在多个长周期基准任务（如家居导航、对话式任务规划）中，相比传统记忆增强模型，任务成功率提升12%-18%，且记忆存储开销降低约30%。局限性在于当前记忆评估规则依赖预定义特征，未来可引入在线学习以自适应任务环境变化。

### 5. Can Dialects Be Steered Like Languages? Sparse Neurons and Distributed Directions in Arabic LLMs
> 链接：https://arxiv.org/abs/2607.03936

**中文摘要：**  
本文探讨了阿拉伯语大语言模型（LLMs）中方言与标准阿拉伯语（MSA）的神经表征差异，核心贡献在于揭示了方言调控的稀疏性与方向性机制。通过激活修补（activation patching）与探针分析，研究发现模型对特定方言的响应依赖于少量稀疏神经元，而非广泛分布的语义方向；同时，方言与MSA在隐藏空间中存在可分离的线性方向，但方言间的区分度较低。实验表明，通过干预这些稀疏神经元或方向向量，可有效引导模型输出特定方言，但方言内部变体（如埃及方言与黎凡特方言）的调控效果弱于跨语言切换。局限性在于方言数据规模有限，且模型对罕见方言的泛化能力不足，未来需扩展多方言语料并探索跨语言迁移的通用机制。
