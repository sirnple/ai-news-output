## 重点推荐

### 1. LongStraw: Long-Context RL Beyond 2M Tokens under a Fixed GPU Budget
> 来源：papers | 标签：论文 / LLM

**中文摘要：**  
本文提出 **LongStraw**，一种在固定GPU预算下实现超长上下文（超过200万token）强化学习训练的高效框架。其核心贡献在于通过**分阶段上下文扩展**与**动态内存管理**策略，在不增加显存开销的前提下，使模型逐步适应并处理极长序列。方法上，LongStraw将训练过程划分为多个阶段，每个阶段逐步增加上下文长度，并利用**梯度检查点**与**混合精度训练**技术优化显存占用。关键发现表明，该框架能够有效缓解长序列训练中的显存瓶颈，使模型在2M token上下文下仍保持稳定的收敛性能，且推理质量与短上下文基线相当。局限性在于当前方法主要针对特定架构（如Transformer）优化，未来可探索更通用的长上下文强化学习范式。

阅读原文：https://arxiv.org/abs/2607.14952

### 2. Token Time Continuous Diffusion for Language Modeling
> 来源：papers | 标签：论文 / LLM

**中文摘要**  
本文提出一种基于连续时间扩散过程的语言建模方法——Token Time Continuous Diffusion（TTCD），其核心贡献在于将离散文本序列建模为连续时间上的扩散过程，从而在生成质量与采样效率之间实现更优平衡。该方法通过设计时间连续的噪声调度与去噪网络，使模型能够灵活控制生成步数，并利用连续时间框架下的概率流常微分方程实现高效采样。关键发现表明，TTCD在语言建模困惑度与文本生成质量上显著优于传统离散扩散模型，同时采样速度提升数倍，且能通过调整时间步长实现生成质量与速度的动态权衡。然而，该方法在长文本生成中仍存在上下文一致性不足的问题，未来可探索结合自回归先验或层次化扩散策略以增强长程依赖建模能力。

阅读原文：https://arxiv.org/abs/2607.14106

### 3. GRASP: GRanularity-Aware Search Policy for Agentic RAG
> 来源：papers | 标签：论文 / Agent

**中文摘要：**

本文提出GRASP（GRanularity-Aware Search Policy），一种面向智能体检索增强生成（Agentic RAG）的粒度感知搜索策略，其核心贡献在于通过动态调整检索粒度来提升复杂多步推理任务的效率与准确性。方法上，GRASP引入一个轻量级策略网络，根据当前查询的语义复杂度和已获取信息的完整度，自适应地选择全局搜索、局部细化或终止检索等不同粒度的搜索动作，从而在避免冗余检索的同时确保关键信息的覆盖。关键发现表明，在多个多跳问答与事实验证基准上，GRASP相比固定粒度或简单自适应基线方法，在检索效率上提升约30%，且最终答案准确率提高5-8%。局限性在于当前策略网络对领域迁移的泛化能力有限，未来可探索结合元学习以增强跨任务适应性。

阅读原文：https://arxiv.org/abs/2607.10463

### 4. From Pixels to States: Rethinking Interactive World Models as Game Engines
> 来源：papers | 标签：论文 / Agent

**中文摘要：**  
本文提出将交互式世界模型重新构想为“游戏引擎”的新范式，核心贡献在于从像素级预测转向状态级建模，以提升智能体在复杂环境中的规划与推理能力。方法上，作者设计了一种基于潜在状态表示的架构，通过解耦视觉感知与动态演化，使模型能够学习环境的结构化转移规则，而非直接预测原始像素。关键发现表明，该状态级世界模型在多个模拟基准任务中显著优于传统像素级方法，尤其在长期规划与零样本泛化场景下表现出更强的鲁棒性与数据效率。然而，当前方法仍依赖于显式状态标注或预定义抽象空间，在完全无监督或高维连续控制任务中的泛化能力有限。未来工作可探索如何自动发现状态表征，并扩展至更复杂的现实世界交互场景。

阅读原文：https://arxiv.org/abs/2607.14076

### 5. DeepLoop: Depth Scaling for Looped Transformers
> 来源：papers | 标签：论文 / LLM

**中文摘要：**  
本文提出DeepLoop框架，首次系统性地探索循环Transformer（Looped Transformers）的深度缩放机制，核心贡献在于揭示了通过循环迭代实现网络深度动态扩展的可行性与性能增益。方法上，DeepLoop将固定深度的Transformer模块重复循环执行，并引入可学习的深度缩放因子与残差门控机制，以平衡计算效率与表示能力。关键发现包括：循环次数与模型性能呈对数线性关系，且深度缩放因子能有效缓解梯度消失与表示退化问题；在语言建模与图像分类任务中，DeepLoop以更少参数达到与深层Transformer相当的性能。局限性在于循环结构增加了推理延迟，且深度缩放因子的最优设定仍依赖任务调参。未来工作可探索自适应循环终止策略与硬件协同优化。

阅读原文：https://arxiv.org/abs/2607.13491
